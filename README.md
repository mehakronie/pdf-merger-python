PDF Merger Using Python

This project is a desktop-based PDF Merger application developed using Python. It allows users to select multiple PDF files and merge them into a single document through a simple graphical interface.

Features
Select multiple PDF files
Merge PDFs into one document
Simple GUI using Tkinter
Save merged file anywhere
Error handling support
Technologies Used
Python
Tkinter
pypdf
File Handling
How to Run

Install required library:

pip install pypdf

Run the program:

python pdf_merger.py
